# CODSOFT

This repository contains projects developed as part of the **CodSoft Web Development Internship Program**.

## Projects

### 1. Online Quiz Maker
A React-based Online Quiz Maker application that allows users to:
- Create quizzes
- Attempt quizzes
- View results instantly
- Interactive and user-friendly interface

### 2. Job Board Portal
A React-based Job Board Portal that allows:
- Employers to post jobs
- Candidates to browse opportunities
- Application management
- Responsive and modern UI

## Technologies Used

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Node.js
- npm

## Installation

```bash
git clone <repository-url>
cd project-folder
npm install
npm start
```

## Features

- Responsive Design
- Modern User Interface
- Component-Based Architecture
- Easy Navigation
- React Functional Components
